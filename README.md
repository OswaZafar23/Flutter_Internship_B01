## Project Description
This repository contains the **Flutter-Internship-B01** project, developed during my Flutter internship.  
The project demonstrates hands-on experience in **Flutter app development**, including UI design, state management, API integration, and deployment.  
It serves as a reference for learning **Flutter basics, widgets, and app architecture**.

## Installation
1. Install VS code
2. Install Flutter SDK
3. Install Dart SDK
4. Install Android Studio
5. Install Git

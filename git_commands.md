# Git Commands Reference

1. git init        - Initialize a new Git repository
2. git clone <url> - Clone a repository from GitHub
3. git status      - Check current status of files
4. git add <file>  - Stage a file for commit
5. git commit -m "message" - Commit changes with a message
6. git push origin main - Push changes to remote main branch
7. git pull origin main - Pull latest changes from remote
8. git branch      - List or create branches
9. git checkout <branch> - Switch branches
10. git merge <branch> - Merge another branch into current
